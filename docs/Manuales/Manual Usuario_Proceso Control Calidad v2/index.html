Manual Usuario_Proceso Control Calidad v2

El presente Manual de Usuario contiene la descripción de la funcionalidad de Control de Calidad tanto para el perfil de Supervisor (Inicio y fin del proceso) como el del usuario que va a realizar el proceso de Control de Calidad.

PROCESO CONTROL DE CALIDAD

Es importante indicar que el proceso de Control de Calidad se puede realizar una sola vez en un determinado trámite y únicamente se podrá realizar para trámites que hayan sido distribuidos en APEX en las fases que se requiera realizar el control de calidad ya que los expedientes retornarán a los usuarios a los que inicialmente fueron distribuidos. **

Para verificar si un trámite ha sido distribuido en APEX se debe utilizar el reporte Detalle Fases Trámite:

![](Aspose.Words.df94c5b9-c2bf-4056-a493-8b8db8ca7289.002.svg)

En el caso de que un trámite ha sido distribuido en una fase determinada se mostrará de la siguiente manera:

En el caso de un trámite que no ha sido distribuido en APEX no se mostrará ninguna información como se indica a continuación:

  1. Apertura del Proceso de Control de Calidad (Supervisor Gestión Documental/ Supervisor Auditoria Médica/ Jefe Auditoria/ Jefe Tarifario)
  • Ingresar a la opción de Consulta de trámites.

  • Seleccionar e ingresar al trámite que se requiere realizar el control de calidad, haciendo clic en la lupa que se muestra al lado izquierdo del número de trámite.

Hacer clic en el botón Control de Calidad del bloque Opciones Generales Supervisión:

La pantalla de Control de Calidad contiene los siguientes bloques:

  • Información del trámite

Se muestra la información del trámite como estado, fecha recepción, prestador y servicio. Adicionalmente se muestra información importante para el Control de Calidad, como:

Fase Actual: Indica la fase en la que al momento se encuentra distribuido el trámite.

Estado Fase: Estado Iniciado Indica que el trámite está distribuido, pero aún no ha sido finalizada la fase, Estado Finalizado Indica que ya se ha finalizado la fase y está listo el trámite para distribuir en la siguiente fase.

Fase Inicio Apertura Control de Calidad: Indica la fase desde la cual se puede aperturar el Control de Calidad, en caso de que la fase en la que se encuentre el trámite esté en estado Inicial no se requiere “abrir” la fase pues el usuario tiene acceso a realizar los cambios en el trámite, por lo cual únicamente se podrá “abrir” el Control de Calidad en las fases anteriores.

  • Información del proceso de Control de Calidad trámite

Forma de Proceso: Indica si se requiere realizar el proceso de Control de Calidad en todo el trámite o sólo en ciertos expedientes.

En caso de que se seleccione la opción por Expedientes se mostrará en la parte inferior el siguiente bloque que permitirá ingresar los expedientes.

Expediente: Se ingresa el número del expediente; se mostrará el listado de los expedientes que vayan cumpliendo con los valores ingresados.

>> Se pasará el expediente ingresado al Listado de Expedientes que se realizará el proceso de Control de Calidad, previa la verificación de que el expediente ingresado corresponda al trámite.

Nota: En caso de que se requiera borrar todo el listado se utilizará el botón Limpiar Listado.

Observación Codificada: Se selecciona una observación del listado que se muestra, su ingreso es obligatorio y se mostrará en la bitácora del trámite.

Observación Complementaria: Se ingresa una observación en caso de que se requiera dar un mayor detalle del proceso, su ingreso no es obligatorio.

Por defecto se muestra la observación: “CONTROL DE CALIDAD DE TODO EL TRÁMITE” o “CONTROL DE CALIDAD DE # EXPEDIENTES.” adicional al ingresado por el usuario.

La observación codificada o complementaria se muestra en la bitácora del trámite en la opción Consulta de Trámites – Observaciones

Fase: Se muestra el listado de las fases en las cuales se puede “abrir” el Control de Calidad. Se pueden seleccionar todas las fases o alguna de ellas, su ingreso es obligatorio.

Luego de ingresar toda la información se presiona el botón Registrar, en caso de que el proceso se ejecute sin novedades se mostrará el siguiente mensaje.

Al hacer clic en el botón Aceptar se permanecerá en la pantalla, pero no se mostrará el botón Registrar, para salir de la pantalla se debe presionar el botón Atrás.

  1. Autorización del proceso de Control de Calidad en las fases seleccionadas (Supervisor Gestión Documental – Supervisor Auditoría Médica/Jefe Tarifario).

Al registrar el proceso de Control de Calidad se autorizará automáticamente la fase a la cual pertenezca el Supervisor/Jefe que realiza el proceso.

Para que los Supervisores/Jefes de las demás fases realicen la Autorización del proceso de Control de Calidad deben ingresar a Consulta Trámites – Seleccionar el trámite y en el bloque Opciones Generales Supervisión presionar el botón Autorización – CC

En esta pantalla se muestra información del Trámite y del Proceso de Control de Calidad que se requiere autorizar.

NOTAS IMPORTANTES:

  • Para que una fase pueda iniciar el proceso de Control de Calidad requiere que la fase anterior haya finalizado el proceso previamente. Todas las fases en las que se aperturó el Control de Calidad podrán visualizar los expedientes, pero no tendrán habilitado el botón de Inicio hasta que la anterior fase haya finalizado el proceso.

  • Adicionalmente es importante indicar que mientras no se autorice el Proceso de Control de Calidad en la fase, al Usuario que procesa los expedientes no se le mostrará el botón Expedientes Pendientes.

En caso de que no se requiera realizar el Proceso de Control de Calidad en una fase se utilizará la Opción Retirar, de tal forma que no se afecte al proceso de Control de Calidad de las siguientes fases:

  1. Procesamiento de los expedientes de Control de Calidad (Gestor Documental/ Auditor Médico/ Liquidador)

Para que los expedientes puedan ser procesados en las respectivas fases por los usuarios se requiere lo siguiente:

  • Se haya realizado la Autorización del Proceso de Control de Calidad de la fase.
  • Se haya finalizado el proceso de Control de Calidad de la fase anterior o la fase anterior haya retirado (no haya autorizado) el Control de Calidad en esa fase.

Los expedientes para realizar el proceso de Control de Calidad se muestran en la opción Trámites – Opción Expedientes Pendientes.

La Pantalla de Proceso de Control de Calidad (Expedientes Pendientes) contiene los siguientes bloques:

Bloque de Información del Trámite

Muestra la información general del Trámite y adicionalmente el número de expedientes pendientes relacionados con el Control de Calidad en cada una de las fases.

Esta información tiene como fin guiar a los usuarios sobre el estado del proceso de Control de Calidad en cada una de las fases.

Bloque expedientes No Distribuidos

En este bloque se mostrarán los expedientes que no han sido distribuidos en Apex y que pueden ser procesados con las mismas funcionalidades de la opción Expedientes (Inicio, Pausa, Retirar Pausa, Finalizar y Editar).

Los expedientes procesados se podrán visualizar en la pantalla de Expedientes Finalizados No Distribuidos.

Bloque expedientes Control de Calidad

En este bloque se mostrarán los expedientes que se requiere realizar el proceso de Control de Calidad con las mismas funcionalidades de la opción Expedientes (Inicio, Pausa, Retirar Pausa, Finalizar y Editar).

Los expedientes que se requiere realizar el Control de Calidad ya fueron procesados por lo cual se mostrarán inicialmente en la pantalla de Expedientes Finalizados CC, y requerirán ser reiniciados (uno a la vez), luego de lo cual se mostrarán en la pantalla de expedientes y podrán ser iniciados, editados y finalizados.

Para distinguir los expedientes que han sido finalizados con Control de Calidad se mostrarán con un color diferente, para que el usuario pueda finalizar el proceso de Control de Calidad se requerirá que todos los expedientes sean finalizados.

  1. Supervisión Trámites Control de Calidad – Fin de proceso de Control Calidad por fase.

La supervisión se podrá realizar en la opción Procesos – Supervisión y Monitoreo – Trámites Control Calidad.

En la pantalla que se muestra se debe presionar el botón Inicio Supervisión.

Se mostrará el listado de los trámites de la fase que están en proceso de Control de Calidad.

En esta pantalla se muestran pintados de color azul los trámites que tienen todos los expedientes definidos para realizar Control de Calidad procesados (inicio-fin).

Al seleccionar el trámite se muestra la siguiente pantalla con el listado de los expedientes que

se definió para realizar Control de Calidad.

En esta pantalla se mostrarán con distinto color los expedientes que han sido supervisados (verde), los que tienen observaciones contestadas (azul) y no contestadas (rosado).

Para revisar el detalle del expediente y enviar observaciones en caso de requerirlo se debe presionar el botón Ver Detalle.

  1. Finalización del proceso de Control de Calidad en la fase.

Es necesario finalizar el proceso de Control de Calidad en una fase para que la siguiente pueda iniciar con el proceso.

Para finalizar el proceso se verifica que el trámite no tenga observaciones pendientes de contestar y expedientes con Control de Calidad pendientes de realizar la revisión.
